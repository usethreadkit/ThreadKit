# ThreadKit Integrations

This directory contains integrations for popular platforms and CMSes.

## Available Integrations

### WordPress
Location: `wordpress/`

> ⚠️ **Status**: In Development - Not ready for production use

WordPress plugin that allows you to easily embed ThreadKit comments on your WordPress site.

**Features:**
- Automatic embedding on posts and pages
- Replace default WordPress comments
- Shortcode support for manual placement
- Admin settings panel

**Development Status**: See [wordpress/STATUS.md](wordpress/STATUS.md) for current status and [wordpress/TODO.md](wordpress/TODO.md) for roadmap.

See [wordpress/README.md](wordpress/README.md) for developer documentation.
