ThreadKit - Vanilla JS Example

ThreadKit can be used without any framework using the @threadkit/core package directly.

// Import ThreadKit core import { CommentStore, WebSocketClient } from '@threadkit/core'; import '@threadkit/core/styles'; // Create a comment store const store = new CommentStore({ siteId: 'demo', url: '/vanilla', apiUrl: '/api', getToken: () => localStorage.getItem('threadkit_token'), }); // Subscribe to state changes store.on('stateChange', (state) => { // Re-render UI with state.comments }); // Fetch comments store.fetch();
