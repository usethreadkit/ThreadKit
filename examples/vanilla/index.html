ThreadKit - Vanilla JS Example

ThreadKit can be used without a build step using the render() function.

// Import ThreadKit import { render } from '@threadkit/react'; import '@threadkit/react/styles'; // Render into a DOM element const threadkit = render('#threadkit-container', { siteId: 'demo', url: '/vanilla', apiUrl: '/api', mode: 'comments', theme: 'light', }); // To unmount later: // threadkit.unmount();
