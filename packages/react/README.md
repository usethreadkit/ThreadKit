# @threadkit/react

Privacy-first commenting and chat system for React.

## Installation

```bash
npm install @threadkit/react
```

## Quick Start

```tsx
import { ThreadKit } from '@threadkit/react';
import '@threadkit/react/styles.css';

function App() {
  return (
    <ThreadKit
      apiKey="your-api-key"
      pageId="unique-page-id"
    />
  );
}
```
